Calibration

Unlike expensive photospectrometer devices the ColorHug is a colorimeter and measures a single set of primaries. Different displays have very different primaries and we can use user contributed matrices to better adapt the ColorHug sensor to a particular display technology.

Using the factory supplied matrices you can get pretty accurate color readings without actually changing anthing. To more accurately measure colors from a display, ccmx files are used. These specific to a particular display make and model combination.

You can update the list of calibration matrices and load new data on to the ColorHug device by running colorhug-ccmx.

Firmware

Unlike proprietary devices the firmware may be updated as new versions are released. Firmware releases will add features, fix bugs and improve accuracy. The client tools can be used to update the firmware easily. The firmware will always be available to download for free.

You can check and update your firmware version by running colorhug-flash.

If you want to play with different versions of the flash code, all the past versions are archived here and can be written to the ColorHug using /usr/libexec/colorhug flash-firmware.

Stay tuned about new firmware and client code updates by subscribing to our newsletter.

Getting the source

You can find all the source code (including the bootloader, firmware, client, schematic, website and PCB) in the several colorhug git repos. It's all licenced as GPLv2+ unless otherwise noted.
