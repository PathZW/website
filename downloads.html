Client code

If you are not running Fedora 16 you may need to compile and build the client tools tarball.

The client tools are required if you want to test or flash new firmware on your ColorHug.

Firmware

Unlike proprietary devices the firmware may be updated as new versions are released. Firmware releases will add features, fix bugs and improve accuracy. The client tools can be used to update the firmware easily. The firmware is available to download free from this website.

You can check your firmware version by typing: 

/usr/libexec/colorhug get-firmware-version

Getting the source

You can find all the source code (including the bootloader, firmware, client, schematic, website and PCB) in the several colorhug git repos. It's all licenced as GPLv2+ unless otherwise noted.
